Current MVA style

MVA Legacy style

To date, My VAC Account has followed Web Experience Toolkit (WET) base theme to guide its design.

The WET base theme is the basis for the MVA Legacy style:

MVA Beta style

In 2019, veterans.gc.ca released a content style guide and a renewed visual theme. As its base, this redesign used the Canada.ca design system for guidance and leveraged GCWeb.

In 2022, we began the redesign of My VAC Account. The 2019 veterans.gc.ca visual theme influenced the design. But, we do not have the GCWeb theme for WET, leading to the following:

WET + GCWeb

In 2022, we also upgraded to the latest version of WET, making the GCWeb theme an option for My VAC Account.

What is WET?

The Web Experience Toolkit is a collection of helpful tools for website interactivity.

You can turn off WET by viewing the page in basic HTML. Just add ?wbdisable=true to any URL.

WET allows us to use pre-built templates and tools.

For instance, one of WET's tools is Datatables which enables interactive HTML tables:

GCWeb also includes some more tools.

  • Do action: Execute action, like filtering a table.
  • URL mapping: Tweak page content based on URL query.
  • Field flow: Build interactive forms.

The benefit of these tools being in the framework is that we build and maintain fewer custom scripts.

According to Who has to use the Canada.ca design system:

"[Veterans Affairs Canada] must apply Canada.ca design requirements for all their public-facing [...] digital services, regardless of the technology, ...or publishing platform..."

Related WET Projects: Centrally Deployed Template Solution (CDTS)

What is the Centrally Deployed Templates Solution (CDTS)?

"The purpose of the CDTS is [...] to deliver the presentation layer of the Canada.ca theme [...] to web assets.

CDTS can help us keep the assets for GCWeb and WET update.

It also provides templates and tools for applications.

Implementing CDTS is a more long-term initiative than adding GCWeb. In the meantime, we can use their samples to guide how we design My VAC Account.

  • CDTS Samples
  • CDTS Home page

Conclusion

As we move forward with the MVA beta and redesign, we should use the GCWeb (Canada.ca) theme. It's usability tested, well-maintained and growing.

Building interactions (alerts for example):

  1. Canada.ca's design system guides interactions (Guidance on alerts)
  2. WET provides the base functionality (WET alert base)
  3. GCWeb theme is applied(GCWeb alert theme)

Writing web content:

  1. Canada.ca's style guide for guidance on writing for the web
  2. veterans.gc.ca style guide for VAC's audience
