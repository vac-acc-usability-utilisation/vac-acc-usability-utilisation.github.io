Switch

Switches toggle the selection of an item on or off

info Overview style Specs design_services Guidelines accessibility_new Accessibility
  • Use toggle switches when the action is binary and should be immediate, like enabling or disabling something instantly.
  • Use checkboxes when the user is making a selection that doesn’t require immediate action and when clarity and accessibility are top priorities.
