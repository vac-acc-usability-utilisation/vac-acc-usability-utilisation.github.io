What is CSA's design system?

The design system includes in-depth UX guidance and UI components for VAC's Client Services Application.

It is based on Material Design 3 and Angular Material, and incorporates guidance from:

  • VAC's style guide and look and feel documentation
  • GCWeb and GCDesign

Portions of this guide are adapted from the Material Design guidelines by Google, available under the Creative Commons Attribution 4.0 License. Changes have been made.

The design guided uses Beer CSS, licensed under the MIT License, to replicate Material Design components.

Using CSA design system

The design system is organized in three main parts: foundations, styles, and components

Foundations

Foundations inform the basis of any great user interface, from accessibility standards to essential patterns for layout and interaction

Styles

Styles are the visual aspects of the UI that give it a distinct look and feel

Components

Components are interactive building blocks for creating a user interface
